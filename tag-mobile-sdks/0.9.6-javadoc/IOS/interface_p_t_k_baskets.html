Detailed Description

Represents the users shopping baskets.

A single shopping basket is maintained per-merchant. A basket is considered to exist - and the list of items in the basket valid - as long as the basket is not empty. Once a basket is emptied adding a new variant for the same merchant will result in the creation of a new basket and you must retrieve a reference to the list of items again.
