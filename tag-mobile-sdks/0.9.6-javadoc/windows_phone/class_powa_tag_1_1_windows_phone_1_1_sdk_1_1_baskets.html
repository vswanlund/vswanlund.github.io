Detailed Description

Represents a public API to keep track of the users current set of originalBaskets.

Any immediate updates to a PersistentBasket object returned will not be reflected in the users Baskets. To persist changes to a Basket call the appropriate BasketsManager API for your use case, the changes will be reflected in the users set of originalBaskets once the operation completes.
