List<ProductOptionChoice> com.powatag.android.sdk.ProductVariantFilter.getOptionChoices ( @NonNull final List< ProductOption >  chosenOptions)

Gets the list of choices and options with the chosen options set according the given options.

The order of the choices will preserve any the order of product options. For a product with "color" and "size" choices, filtering for a particular color (e.g. red) will return the color choice with the chosen option as red and the possible size options available for that color. Will return an empty list if the product has single variant.
