Detailed Description

Class of supported countries.

Represents an ISO 3166 country. For more information on ISO 3166 country codes visit http://www.iso.org/iso/home/standards/country_codes.htm. A table of the latest codes can be found at https://www.iso.org/obp/ui/#search.

Method Documentation

+ (nonnull instancetype) countryForCode: (nonnull NSString *)  alpha2Code

Gets a country for the specified alpha 2 code, returning one of the pre-defined supported country constants if the code matches one of them, or a new country object with the specified alpha 2 code otherwise.

Parameters
alpha2CodeThe ISO 3166-1 alpha-2 country code.
Returns
