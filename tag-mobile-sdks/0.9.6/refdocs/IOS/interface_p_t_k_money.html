Detailed Description

An amount of money with the decimal places defined by the currency.

This class represents a quantity of money, stored as a

NSDecimalNumber

amount in a single currency unit. Every currency has a certain standard number of decimal places. The

Money

class is fixed to this number of decimal places. Calculations on fractions of a cent are not permitted by this class. This class is immutable and thread-safe.
