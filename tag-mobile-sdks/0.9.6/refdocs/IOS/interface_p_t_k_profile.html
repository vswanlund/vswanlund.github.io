Detailed Description

Represents a basic PowaTag user profile.

To update the profile use editableCopy to create an PTKProfileDetails that can be modified.

Method Documentation

Parameters
merchantThe merchant these accepted methods are for.

The list of payment instruments for the user which are accepted by the specified merchant, as defined by {@link PTKMerchant::paymentInstruments}, or an empty list if none of the users payment methods are accepted.
