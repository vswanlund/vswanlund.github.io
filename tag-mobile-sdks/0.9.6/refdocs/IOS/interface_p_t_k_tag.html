Detailed Description

Represents a PowaTag, such as a QR code, Audio Tag or Beacon.

A tag is uniquely identified by a reference and format. Two tags are considered equal if both their reference and format are the same. A convenience method is provided for parsing tags from URL strings and audio, see parseQrUrl:error: or parseAudio:error:. This class is immutable and thread-safe.
