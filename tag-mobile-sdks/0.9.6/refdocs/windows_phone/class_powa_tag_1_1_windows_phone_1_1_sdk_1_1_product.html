Detailed Description

A Product, such as a T-Shirt.

Information such as SKU (stock keeping unit) are stored at the product variant level. Each product has at least one variant (e.g. small T-Shirt). If a product has options it may have more than variant (corresponding to the valid combinations of option choices).

This class is immutable and thread-safe.
