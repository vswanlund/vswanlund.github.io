- (BOOL) isDetecting

Whether the detector is currently detecting tags and recording audio from the microphone.

Returns
true
if the detector is currently detecting,
false
otherwise.
- (void) startDetection

Starts recording audio from the microphone and searching for embedded tags.

Has no effect if detection has already been started.

- (void) stopDetection

Stops detecting tags and recording audio from the microphone.

Has no effect if detection is not currently ongoing.
