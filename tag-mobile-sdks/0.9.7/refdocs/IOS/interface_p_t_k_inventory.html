Detailed Description

A description of the availability of a product variant.

You should first check whether the variant {@link PTKInventory::forSale}, provided it is this should be followed by checking if the variant {@link PTKInventory::unlimitedStock}, in which case the quantity and back order eligibility can safely be ignored. If the variant does not have unlimited stock the {@link PTKInventory::quantity} will return the current stock level. Even if the quantity is zero the product may still be purchased if it {@link PTKInventory::eligibleForBackOrder} though this will usually take longer to deliver than a standard order and the user should be made aware of this before purchasing. This class is immutable and thread-safe.

The quantity of the product variant currently available in stock.

This value can safely be ignored if the variant {@link PTKInventory::unlimitedStock}. It may still be possible to purchase the variant even if there is no currently available stock if the variant {@link PTKInventory::eligibleForBackOrder}.
