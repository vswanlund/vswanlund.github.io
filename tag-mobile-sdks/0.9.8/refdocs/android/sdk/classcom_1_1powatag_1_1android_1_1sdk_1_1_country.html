Detailed Description

Class of supported countries.

Represents an ISO 3166 country. For more information on ISO 3166 country codes visit http://www.iso.org/iso/home/standards/country_codes.htm. A table of the latest codes can be found at https://www.iso.org/obp/ui/#search.
