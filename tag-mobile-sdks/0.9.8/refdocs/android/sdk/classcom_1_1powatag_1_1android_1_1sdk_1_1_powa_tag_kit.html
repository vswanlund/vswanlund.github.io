Initializes the PowaTag SDK library, the behavior of SDK functions are undetermined if this function is not called.

It should be called as early as possible, ideally in the

onCreate()

method of your

Application

. It is safe to call this method multiple times, if called more than once subsequent calls have no effect.

Parameters
applicationContextThe application context.
powaTagHostNameThe host name, not including the port, e.g. "https://powatag.com". to use for managers and services instead of the default host name "https://powatag.com" and ports. The endpoint should never be changed in production code.
apiKeyThe API key for the application.
secretThe secret for the application.
