boolean com.powatag.android.sdk.ProfileCacheManager.loadCurrentProfile ( )

Loads a cached profile and sets it as the current profile.

If no profile was cached the current profile will be unchanged. Should be called when the SDK is initialized. Loading the profile from the cache may take some time so this should be performed in the background if possible.

Returns
true
if a profile was loaded.
